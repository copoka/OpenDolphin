
OpenDolphin のコンパイルについて

　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　2004年6月15日
　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　（株）デジタルグローブ

　
　
OpenDolphin は Java プログラムです。下記環境でコンパイルすることができます。

１．SDK  Java 2 Platform, Standard Edition, v 1.4.2 以降
２．コンパイルにはダウンロードした ext ディレクトリに内にある Jar ファイル全てが必要です。
　　これらを sdk の jre/lib/ext 以下へコピーするか、全ての jar ファイルに classpath を通すか
　　してください。
